# MAGNA Microsite - PRD

## Problema Original
Desarrollar un microsite exclusivo para la línea MAGNA de Morelli que funcione como pieza clave de posicionamiento y herramienta comercial, transmitiendo su carácter premium y generando consultas directas.

## Concepto Creativo
**"Silenciosamente Imponente"** - MAGNA es autoridad, precisión y carácter en estado puro. No necesita exagerar: su presencia transforma el espacio. El lujo verdadero no grita. Se siente.

## Target Audience
- Público aspiracional que busca cocina premium con respaldo profesional
- Arquitectos y diseñadores de interiores
- Clientes que valoran diseño atemporal, rendimiento real y estatus

## Implementado ✅
- **Hero Section**: Diseño cinematográfico con imagen de fondo, título MAGNA y subtítulo "Silenciosamente Imponente"
- **Manifiesto**: Sección con storytelling emocional sobre la marca
- **Productos**: Showcase de MAGNA 1200 y MAGNA 900 con características
- **Especificaciones Técnicas**: Grid Bento con tabs para cada modelo (dimensiones, potencia, materiales, características)
- **Galería**: Grid de imágenes premium de cocinas
- **Formulario de Contacto**: Captura de leads con envío a valeria@wtf-agency.com
- **Selector de Idioma**: ES/PT completamente funcional
- **Backend API**: Endpoint /api/contact para almacenar leads en MongoDB + envío de emails via Resend
- **Diseño Premium**: Tema oscuro (#050505), tipografías Italiana + Manrope, animaciones suaves

## Stack Técnico
- Frontend: React + Tailwind CSS + Framer Motion + Shadcn UI
- Backend: FastAPI + MongoDB + Resend
- Idiomas: Español y Portugués
